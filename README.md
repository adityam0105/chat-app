# CHAT-APP

A standard MERN Stack Chat App

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JavaScript
- Tailwind CSS
- DaisyUI
- JWT (JSON Web Tokens)
- Socket.io
- bcrypt.js

## Description

The application uses consistent routing and authentication with distinct Webpages on which the users can Signup,Login,

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Run the froentend server using `npm run dev`.
5. Run the backend server using `npm run server`.
6. Create a global .env file containing the MONGO_DB_URI, PORT and the JWT_SECRET


## Features

- Sending and Recieving Messages 
- A Secure Chat App for a small community of friends

## Authentication

This project uses JWT (JSON Web Tokens) and Socket.io for authentication and bcrypt.js for password hashing.
